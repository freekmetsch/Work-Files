# Learning Outcomes → Test Mapping Analysis

**Purpose:** Map Kennisbasis requirements to learning outcomes, then determine which test component validates each outcome.
**Status:** v3 - Voice notes integrated (2025-01-29)

---

## 1. KENNISBASIS 2.2 PRODUCTIEVE VAARDIGHEDEN

### Core Concept
> De startbekwame leraar produceert uiteenlopende vormen van taaluitingen in de doeltaal op **een gemiddeld C2 niveau** van het ERK.

### 2.2.1 Clear & Coherent Writing
- Helder en samenhangend schrijven
- Passend bij **communicatieve doel** (communicative purpose)
- Passend bij **doelgroep** (target audience)
- Passend bij **context**

### 2.2.2 Productive Strategies
- Schrijven over **vertrouwde en niet-vertrouwde onderwerpen** (familiar + unfamiliar)
- **Inzetten van passende schrijfstrategieën** om kwaliteit te verbeteren
- **Inzetten van conventies passend bij genre**
- Consistente en nauwkeurige grammaticale beheersing; fouten nagenoeg afwezig

## 3. DESIGN PRINCIPLES FOR NEW LEARNING OUTCOMES

### Decisions (2025-01-29):
1. **APA + Source Integration = ONE outcome** ("integrate sources using APA")
2. **Audience awareness + Genre = ONE outcome** (audience is part of genre conventions)
3. **Grammar stays implied** via CEFR level (no separate outcome needed)
4. **Use "author" as a verb** — captures the validity/ownership point beautifully
5. **Strategy progression:** Same core verb, increasingly intense adverb
6. **Outcome 1 tiered:** Awareness level progresses across courses
7. **Familiar/unfamiliar topics:** Incorporated into Outcome 1

### Outcome 1 Progression

W1, W2, and W3 all share the same core outcome structure for communicative purpose, audience, and genre conventions, with the level of awareness increasing: W1 uses "reflect awareness," W2 uses "demonstrate deliberate application," and W3 uses "strategically leverage."

### Strategy Progression (Outcome 3)

The strategy outcome uses the same core verb with increasingly intense adverbs: W1 uses "apply and justify," W2 uses "consciously apply and justify," and W3 uses "strategically apply and justify."

### Authorship Principle:
> "Author" captures what we want students to demonstrate: ownership, intentionality, ability to explain. The P+Q&A validates authorship not as a separate criterion but as the underlying purpose of the assessment format.

## 4. PROPOSED NEW LEARNING OUTCOMES

### Writing 1 (B1/B2)

| # | Learning Outcome |
|---|------------------|
| 1 | Can **author** texts on familiar and unfamiliar topics **considering** of communicative purpose, audience, and genre conventions |
| 2 | Can **integrate sources using APA** referencing |
| 3 | Can **apply and justify** writing strategies appropriate to the text |
| 4 | Can write at **CEFR level B1/B2** |

**Assessment structure:**
- **Timed test (60 min):** Two paragraphs, two genres, source integration (given sources), one familiar topic + one unfamiliar (requires reading source)
- **Take-home:** Article for semi-formal teacher journal — personal, argumentative, professional audience, more structural freedom than formal expository essay (previous task)
- **P+Q&A:** Validates authorship through explanation of purpose, audience, strategies

## 5. TEST COMPONENT → OUTCOME MAPPING

### Admissibility Check (W1 only - Go/No-Go)

Before P+Q&A can proceed:
- **Formatting:** Document properly formatted
- **Language level:** Passes B2 "gut check" (no major errors)
- **APA reference list:** Present for sources used

**Purpose:** Ensures student took assessment seriously and meets minimum threshold for oral defense.

### Timed Test (W1 and W3 only)

**Writing 1 Timed Test (60 min):**
- **Part A:** Familiar prompt (formal/professional, 15-20 min)
  - Examples: Admin email, parent response, colleague memo
- **Part B:** Unfamiliar prompt (expository with sources, 40 min)
  - Example: Summary paragraph on given topic with 1-2 sources
  - Must include 1 quote, 1 paraphrase, APA in-text citations

**Writing 1 Timed Test Rubric (3 criteria):**

| Outcome | Criterion | How Timed Test Addresses It |
|---------|-----------|----------------------------|
| 2. Integrate sources using APA | **Source Integration** | ✅ Directly assessed: quotes grammatically integrated, paraphrases accurate, APA formatting |
| 1. Author texts with awareness... (partial) | **Communicative Effectiveness** | Shows genre application: does text achieve purpose for audience? |
| 4. CEFR level B1/B2 | **Language Level** | ✅ Primary validation: vocabulary, grammar, fluency |

**Writing 3 Timed Test (90 min):**
- Three paragraphs (vs. two in W1)
- 3 sources provided (vs. 1-2 in W1)
- Higher complexity; same rubric structure with C1/C2 descriptors

### P+Q&A (All courses)

**Writing 1 P+Q&A Rubric (2 criteria + admissibility):**

| Outcome | Criterion | How P+Q&A Addresses It |
|---------|-----------|------------------------|
| 1. Author texts with awareness... | **Genre Awareness** | ✅ Validates through articulation: communicative purpose, audience (teachers), genre conventions (content + structure) |
| 3. Apply and justify strategies | **Strategy Application** | ✅ Validates through articulation: high-level process + detailed strategies with examples from text |

**Q&A Format:** Interrogation of claims made in presentation; follows up on elements not yet covered; uses criteria to guide questioning.

**Authorship Note:** Not a separate criterion. If student can explain genre awareness and strategies, they've demonstrated authorship. The P+Q&A format itself validates authorship.

**Writing 2 & 3 P+Q&A:**
- W2: Same 2 criteria but **higher bar for articulation** (no timed test validates language)
- W3: 4-criteria framework (Authorship, Strategy, Decision-Making, Genre) - general enough to apply to both proposal AND creative options

## 6. RUBRIC DERIVATION

### Timed Test Rubric (W1 and W3)

Derived from what timed test validates:

| Criterion | Outcome Link | What It Captures |
|-----------|--------------|------------------|
| **Language Level** | Outcome 4 | Vocabulary, grammar, fluency at target CEFR |
| **Source Integration** | Outcome 2 | Appropriate use of given sources, APA formatting |
| **Structure & Coherence** | Outcome 1 (partial), 3 (implicit) | Paragraph structure, logical flow, genre conventions |

### P+Q&A Rubric (All courses)

Derived from what P+Q&A validates:

| Criterion | Outcome Link | What It Captures |
|-----------|--------------|------------------|
| **Genre & Audience Awareness** | Outcome 1 | Can articulate conventions, audience, communicative purpose |
| **Strategy Articulation** | Outcome 3 | Can explain writing process, name strategies, give examples |
| **Decision Justification** | Outcome 1, 3 | Can explain choices made and alternatives discarded |

**Note on Authorship:** Not a separate criterion. If a student can articulate genre awareness, strategies, and decisions, they've demonstrated authorship. The P+Q&A format itself is what makes authorship visible.
