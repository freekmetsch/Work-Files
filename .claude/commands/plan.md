---
description: Turn raw work ideas into a practical next-step plan
---

> See `/.claude/commands/common.md` for shared context.

Primary use: idea dumps, prioritization, and shaping the best next action.

Constraint: structure first. Do not overplan. Hand off to `/go-work` when the plan leads to material edits.

## 0. Communication
Follow the shared communication rules in `/.claude/commands/common.md`.

Use:
- `No Decision Needed` when the next move is clear,
- `Decision Required` only when a real choice blocks useful planning.

## 1. Input Detection
Choose the mode that fits the input:

- `capture` = raw dump, mixed ideas, vague to-do list
- `shape` = one clear task that needs a workable plan

## 2. Capture Mode
Turn the input into:
- `Goal`
- `Now`
- `Later`
- `Questions`
- `Recommendation`
- `Next`

Prioritize:
- urgency,
- usefulness,
- dependencies,
- clarity of next step.

## 3. Shape Mode
Turn the input into:
- `Goal`
- `Constraints`
- `Scope`
- `Steps`
- `Recommendation`
- `Next`

Keep steps short and concrete.

## 4. Quality Check
Before finalizing, check:
- is the next move concrete,
- are duplicates merged,
- is anything overplanned,
- is the recommendation useful now.

## 5. Output

### Capture Mode
1. `Goal`
2. `Now`
3. `Later`
4. `Questions`
5. `Recommendation`
6. `Next`

### Shape Mode
1. `Goal`
2. `Constraints`
3. `Scope`
4. `Steps`
5. `Recommendation`
6. `Next`

## 6. Hard Stop
End once the next move is clear.
